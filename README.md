# Project: 6th National English Language Carnival 2025 Website

## 1. Overview

This repository contains the source code for the promotional single-page website for the "6th National English Language Carnival 2025," hosted by the Ideal English Language Club (IELC). You can see this site in action at https://6thnelc.saqinnoor.dpdns.org/.

## 2. File Structure

The project consists of the following files:

-   `index.html`: The main and only HTML file containing all the structure, styling, and scripts.
-   `IELC-logo-black.svg`: The high quality logo image used for the header, loader, and favicon.
-   `README.md`: This file.

## 3. Technologies Used

-   **HTML5**: For the core structure and content.
-   **CSS3**: For all styling, animations, and responsiveness. The project uses modern CSS features like CSS Variables, Flexbox, Grid, and keyframe animations. All CSS is self-contained within the `<style>` tag in `index.html`.
-   **Vanilla JavaScript**: For dynamic features like the page loader, scroll-reveal effects, and the mobile navigation menu.

No external frameworks or libraries are used.

## 6. Code Overview

The entire website is contained within `index.html`.
